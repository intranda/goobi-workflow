What to use?

Use <form:field> for any field that contains a single input element.

Use <form:field group="true"> for any field that contains one if the following input types: radio, checkboxes. Also use this for multiple input elements in one field (see the "Assigned rights" as well as the following "Name" field above). This will produce the field as a HTML fieldset, as well as the provided label as a HTML legend instead of a HTML label, to improve accessibility.

Use <form:fieldset> for any field that contains multiple grouped, distinct input elements. Fieldsets may be nested, as shown above.
